# Changelog

## 2026.7.1 - Pre

Initial release.

### Added

- LLM-driven Scratch project generation from natural language descriptions
- SVG rendering engine with support for shapes, gradients, filters, paths, and transforms
- Sprite templates: circle, rectangle, star, arrow
- Backdrop templates: solid color, grid
- 30+ Scratch block types across all categories (motion, looks, events, control, sensing, operators, variables, data)
- `.sb3` packaging and extraction
- Extension system (`.editor` files) for adding blocks and TurboWarp custom extensions
- Extension manager with terminal UI (`--editors`)
- TurboWarp advanced settings extension (framerate, interpolation, pen quality, stage size, clone limit, turbo mode, drag mode)
- LLM API configuration via interactive setup, `config.yaml`, and environment variables
- Block reference from `agent/tools.md` fed to LLM as context
- Internationalization (English and Chinese)
